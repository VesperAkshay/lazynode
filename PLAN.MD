### Keyboard Shortcuts (Lazygit-inspired)
- `j/k` or arrows: Navigate lists
- `enter`: Run selected script
- `i`: Install package
- `d`: Delete package
- `o`: Check outdated
- `u`: Update selected package
- `r`: Reload UI
- `e`: Edit package.json
- `q`: Quit
